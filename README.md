# 🀄️ 麻將戰情室 (Mahjong Helper)

一個專為麻將玩家設計的「戰情室」風格決策助手，提供即時的攻防建議。支援台灣麻將 (16張) 與常見麻將規則。

## ✨ 主要功能

### 1. 戰情室佈局 (War Room UI)
- **四家視角**：清楚顯示上家、對家、下家與自己的捨牌區（河）。
- **雙軌決策 HUD**：
  - **🛡️ 防守面板**：即時列出場上「現物」（絕對安全牌），防止放槍。
  - **⚔️ 進攻面板**：根據手牌效率，分析並建議打出哪張牌（標示生張/熟張）。

### 2. 智慧輔助
- **剩餘張數計算**：自動扣除場上可見牌與自己手牌，計算剩餘張數。
- **孤張識別**：自動識別手牌中效率最低的孤張。
- **危險度提示**：打出建議牌時，會提示該牌是否為生張（危險）或熟張（安全）。

### 3. 操作流程
為了跟上打牌速度，設計了極簡的操作流：
1. **初始配牌**：使用下方鍵盤快速輸入起手牌。
2. **記錄場況**：
   - 點擊「對家/上家/下家」按鈕。
   - 點擊牌面 -> 自動飛入該玩家河底。
3. **自己回合**：
   - 點擊「摸牌」。
   - 參考 HUD 建議。
   - 點擊手牌打出。

## 🚀 快速開始

### 本地運行
直接打開 `index.html` 即可使用。

### 手機使用
建議使用 GitHub Pages 部署，並將網頁「加入主畫面」以獲得最佳全螢幕體驗。

## ⚠️ 注意事項
- 本工具僅供**練習與輔助**使用。
- **象棋麻將**模式目前尚未實裝，目前僅支援索/筒/萬/字牌模式。
